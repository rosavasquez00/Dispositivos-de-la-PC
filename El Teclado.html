El Teclado

El teclado es el dispositivo más común de entrada de datos. Se utiliza para introducir comandos textos y números. El teclado típico contiene tres partes: teclas de función, teclas alfanuméricas, y el subteclado numérico. Las teclas de función (F1, F2, F3…,) son un conjunto de teclas que se reservan para realizar funciones especiales. Dichas teclas se utilizan de modo distinto por los diferentes programas existentes en una instalación computacional. Las teclas alfanuméricas sirven para ingresar caracteres alfanuméricos, numéricos y de puntuación. El subteclado numérico permite ingresar rápidamente números y símbolos aritméticos para efectuar cálculos matemáticos. El cursor puede ser manipulado utilizando algunas de las teclas del subteclado numérico.
